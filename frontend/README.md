# Government of Agrabah Services Portal - Frontend

App for managing utility services (Electricity, Water, Gas) with bill payments, service requests, and customer support.

- Frontend template: https://github.com/NagariaHussain/doppio_frappeui_starter
- Other libraries: tanstack-query (for vue)


##  Folders
```
src/
├── api/              # backend API communication
├── components/       # reusable vue components
├── composables/      # vue composables
├── constants/        # data constants
├── data/             # data stores
├── pages/            # page views
├── utils/            # utility functions
├── assets/           # static assets 
```

## Setup
```bash
cd apps/agrabah_portal/frontend
yarn install
yarn dev
```
